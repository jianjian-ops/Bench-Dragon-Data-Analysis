板凳龙运动轨迹数据分析项目

## 项目背景
本项目以传统民俗活动“板凳龙”为研究场景，
将舞龙队运动过程抽象为多节点、多时间点的运动数据分析问题。
通过数据计算、仿真与可视化分析，
评估盘入过程中的安全边界，并为路径与速度优化提供数据支持。

## 分析目标
- 计算舞龙队在不同时间点的节点位置与速度数据
- 分析盘入过程中可能出现的风险临界状态
- 对路径参数进行对比分析，评估其对运动效果的影响
- 通过数据结果支持路径与速度的优化决策

## 方法与思路
- 使用 Python 构建运动轨迹计算模型，生成多节点时序数据
- 对计算结果进行批量处理，并输出为 Excel 数据文件
- 设计异常检测逻辑，用于识别潜在在碰撞风险
- 对结果进行可视化分析，展示运动轨迹与速度变化趋势

## 使用工具
- Python（Pandas、NumPy、Matplotlib）
- Excel（结果整理与分析）
- GitHub（项目管理与展示）

## 项目成果
- 生成 223 节板凳在多时间点的运动数据
- 识别盘入过程中的关键风险时刻
- 对不同参数条件下的运动结果进行对比分析
- 输出结构化数据结果与可视化图表

## 项目说明
本项目为数据分析与建模实践项目，侧重数据处理、分析逻辑与结果表达。
